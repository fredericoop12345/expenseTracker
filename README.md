This project was done with the help of the platform https://roadmap.sh/projects/expense-tracker 
This project practises working with CLI arguments by developing an Expense tracker.


The following is the command to type to run the help instructions:
 expense-tracker showhelp

Requirement:
Have the latest version of python installed.

Type this in the terminal to open the .bash_profile
    vi ~/.bash_profile

Press i to enter normal mode

Type the following on a line:
    alias expense-tracker="python3 /Users/fredrickoppong/expenseTracker/expenseTracker.py"

Type ':' to the screen followed with wq and press the Enter key.

Type 'source ~/.bash_profile in the terminal before running the CLI Interface
